В каталоге находятся исходники диаграмм в формате `draw.io`:
- `IDEF0_A-0.drawio`
- `IDEF0_A0.drawio`
- `IDEF0_A2_decomposition.drawio`
- `IDEF3.drawio`
- `DFD.drawio`
- `UML_Use_Case.drawio`
- `UML_Activity.drawio`
- `UML_Sequence.drawio`
- `IDEF1X.drawio`

Экспорт в PNG:
1. Открыть файл в diagrams.net (draw.io).
2. Меню `File -> Export as -> PNG`.
3. Сохранить PNG рядом с исходником для вставки в ПЗ и презентацию.
